# Cryptofy

Cryptofy is a real-time cryptocurrency tracking application that integrates with a public API to fetch live data. Users can click on a cryptocurrency to view detailed information along with a chart made using Google React Charts.

## Features
- Real-time cryptocurrency data
- Interactive charts using Google React Charts
- Detailed view for each cryptocurrency
- Responsive design

## Technologies Used
- React.js
- Tailwind CSS
- API Integration for real-time data
- Google React Charts

## Installation

Follow these steps to set up Cryptofy on your local machine:

```bash
# Clone the repository
git clone https://github.com/your-username/cryptofy.git
cd cryptofy

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Build for Production

```bash
npm run build
```
